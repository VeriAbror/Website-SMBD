Topik 1: Instalasi dan Konfigurasi Server Database

Latar Belakang

Dalam pengelolaan data, server database menjadi komponen penting untuk menyimpan, mengelola, dan mengakses data secara efisien. Instalasi dan konfigurasi server database seperti MySQL adalah langkah awal yang harus dilakukan untuk memastikan sistem berjalan dengan baik.

Problem

Banyak pengguna menghadapi kesulitan dalam proses instalasi dan konfigurasi server database, terutama dalam memahami pengaturan awal seperti user management, port configuration, dan pengaturan keamanan.

Database Relational dan Unrelational

Database Relational

Database relational menyimpan data dalam bentuk tabel dengan hubungan antar tabel menggunakan kunci (key). Contoh produk:

  • MySQL
  • PostgreSQL
  • Microsoft SQL Server

Database Unrelational (NoSQL)

Database unrelational menyimpan data dalam format non-tabular seperti dokumen, key-value, atau graph. Contoh produk:

  • MongoDB
  • Redis
  • Cassandra

Kapan Menggunakan Relational dan Unrelational

Gunakan Database Relational Jika:

  • Data memiliki struktur yang jelas (tabel dengan kolom tetap).
  • Membutuhkan integritas data yang tinggi (misalnya transaksi keuangan).
  • Membutuhkan hubungan antar data (relasi antar tabel).

Gunakan Database Unrelational Jika:

  • Data tidak terstruktur atau semi-terstruktur (misalnya JSON, XML).
  • Membutuhkan skalabilitas tinggi untuk data besar.
  • Membutuhkan performa tinggi untuk operasi baca/tulis cepat.

Solusi/Skenario Aktivitas

Berikut adalah langkah-langkah instalasi dan konfigurasi server database MySQL:

  1. Unduh installer MySQL dari situs resmi.
  2. Jalankan installer dan ikuti wizard instalasi.
  3. Konfigurasikan user root dan password.
  4. Atur port default (3306) dan pengaturan keamanan.
  5. Verifikasi instalasi dengan menjalankan perintah mysql --version.

Pembahasan

Setelah instalasi selesai, langkah berikutnya adalah mengakses server database menggunakan command line atau GUI seperti MySQL Workbench. Berikut adalah contoh perintah untuk membuat database baru: 

CREATE DATABASE kelompok_AB_nama_mhs;
USE kelompok_AB_nama_mhs;
CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(100)
);
